Kivy is an open source Python framework, which allows the development of multi-touch mobile and desktop applications with 
Natural User Interface. A designing language, called Kivy Language, to design custom widgets.
Following are the instructions for installation:
Install the kivy:

Step1: sudo add-apt-repository ppa:kivy-team/kivy
Step2: sudo apt-get update
Step3: sudo apt-get install python-kivy

Clone the project:

Step4: git clone https://github.com/Rupinderwadali/Kivy-notification..git
Step5: cd Kivy-notification/
Step:6 python alarm.py

It shows a pop-up. Fill the time and message and click on button to show the alert message. The work is in progress. So change the 
minutes with little difference to show the demo :P.
